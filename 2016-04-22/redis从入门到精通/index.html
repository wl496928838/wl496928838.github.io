安装 redis

apt-get install redis-server

nohup redis-server &

自增

统计活跃度

月活跃度

统计

bitcount string:count:huoyue:user:1

setbit 实际上设置的是内存地址的字节
8位
00000000
setbit key 字节位置 1或者0

bitcount 统计内存地址中数据有多少个1
